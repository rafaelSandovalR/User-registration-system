Welcome to "In My Shoes" a student mentor/student mentee website created to help students connect with each other for general help at college. It has become redundant in our ongoing global pandemic to expect students to show up to campus for issues that can be solved virtually. This is where our website comes to play, by using IMS students can chat/video call to ask questions related to college. They should be able to communicate what they want and be guided to the resource if applicable.

Authors: Sidra Naru, Ryan Setaruddin, Yvens Pierre, Gary Soza, Brain Guerra, and Rafael Ramirez
